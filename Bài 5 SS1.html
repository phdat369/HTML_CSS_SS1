Giới thiệu về môn Frontend

Frontend là gì ?

Frontend là phần quan trọng của phát triển web , bao gồm mọi thứ mà người dùng nhìn thấy và tương tác trực tiếp trên web

Các Công Nghệ Frontend Phổ Biến

  • HTML : Ngôn ngữ đánh dấu siêu văn bản , là nền tảng để xây dựng cấu trúc của trang web
  • CSS : Ngôn ngữ để định dạng và tạo kiểu cho trang web , giúp trang web trở nên hấp dẫn và dễ sử dụng
  • JS : Ngôn ngữ lập trình làm cho trang web trở nên động , với các tính năng như xử lí sự kiện , hiệu ứng động và xử lí máy chủ

Vai Trò của Frontend

Frontend giúp tạo ra một trải nghiệm người dùng mượt mà dễ tiếp cận .Những nhà phát triển frontend đảm bảo trang web tương thích trên các thiết bị và trình duyệt khác nhau
